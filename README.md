# Deteccion-de-personas

Este es el código usado para el análisis de deteccion de personas con el uso de partes del cuerpo.

El repositorio consiste de dos notebooks: Entrenamiento y medirresultados. El primero contiene todo respecto al entrenamiento del modelo YOLO y guardar tanto sus predicciones como el modelo en sí. Y el segundo consiste en el análisis de esos resultados.

Para poder correr el código es necesario tener un dataset con las anotaciones adecuadas en un google drive propio, este proyecto ha usado el siguiente: https://drive.google.com/drive/folders/10R8CQkW7dsMDK8qr3XX8SEglrtHAb9IB
